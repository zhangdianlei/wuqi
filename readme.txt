特别说明: (用户必读!)

  + 您使用本系统前,请详细阅读用户授权文档 licence.txt.切记!在您认为可以满足授权文档
    中列出的要求时,方可使用 mymps,否则,请不要将 mymps 安装到您的网络上.

  + 如果您不是从官方得到的本程序,很可能获得的不是最新版本并且无法使用或有很多BUG.
  
  + 请确认您的空间默认首页文档格式排列顺序为index.html(index.htm),index.php

  + 官方推荐服务器配置：php 5.0以上，mysql 5.0以上，apache 2.5以上，并且支持<?=?>短标记格式

  + 请确认以下目录或文件为可写，linux主机下属性为0777可写：

    /data
    /data/sessions
    /data/caches
    /backup
    /attahcment
    /data/config.php
    /data/config.db.php
    /data/membertpl.inc.php
    /data/document.inc.php
    /data/info.type.inc.php
    /data/cron.cache.php
    /attachment

简单的安装方法: (以下每一步都非常关键,请仔细阅读)

  + 上传目录下所有程序文件到服务器

  + 运行 /install 根据提示安装程序,设置管理员账号
  
  + 系统安装完毕后，请删除install目录，然后进入后台管理进行系统设置，
  然后进行行业类别设置，地区设置，然后再进行其它操作。

  + 如果提示您的 config.db.php 无法在线编辑,请在本地编辑好后用 FTP 上传

  + 系统后台管理路径为：http://你的域名/admin/index.php
